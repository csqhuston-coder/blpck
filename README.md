# BLPCK

A living super app. Generative, self-evolving, meeting every human where they are.

## What this repo is right now

The first-touch canvas. The soul layer.

- Living iridescent field rendered in WebGL
- Polyrhythmic base motion (9s, 14s, 17s primes — never align)
- 15-palette color library rotating every 25-40 seconds
- Chaos engine — 9 effect types (chroma, mirror, warp, strobe, invert,
  tessellate, datamosh, scanline, colorburst) with randomized composition
- Butterfly memory — touches persist and shape the field going forward
- iPhone Safari first, but works on everything

## Architecture

- `index.html` — single-file build. WebGL base + Canvas2D overlay + CSS filter layer
- Hosted on Vercel, deploys from main branch on commit
- No build step. No dependencies. Pure web.

## Principles

- BLPCK is alive. Nothing about it is static.
- Every screenshot is a fossil.
- The platform does not announce itself.
- Chaos is the response, not the base.
